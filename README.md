<h1>Grass Renderer - Unity GPU Instancing</h1>

*Generates and renders millions of small 3D objects in real time.*

**<ins>Features</ins>**
- GPU Instancing for real time rendering
- Compute Shader for blade positions and wind animation
- Customizable map size, grass density, grass model, heightmap, wind speed/direction
